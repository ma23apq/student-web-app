# Student Web App (Full Stack CRUD)

## 🚀 Project Overview
This is a full-stack web application built using Flask and SQLite.  
It allows users to add, view, and delete student records through a simple web interface.

---

## 🛠️ Tech Stack
- Python
- Flask
- SQLite
- HTML (Jinja2 Templates)

---

## ✨ Features
- Add new students
- View all students
- Delete students
- Persistent storage using SQLite database

---

## 📂 Project Structure

app.py
templates/
index.html
